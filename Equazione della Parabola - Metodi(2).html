Come determinare l’equazione della parabola

In questa pagina vengono spiegati altri metodi per determinare l'equazione di una parabola in diversi casi: noti punti notevoli o punti appartenenti alla parabola, oppure altre caratteristiche.

1. Noti vertice e direttrice

Per risolvere il problema in questo caso, si procede impostando un sistema matematico con le formule delle coordinate dei punti:

    { -b/2a

    { -Δ/4a

    { (- 1 + Δ)/4a

Una volta averlo risolto e aver trovato tutti i parametri, l'equazione sarà completa.

2. Noti 2 punti e asse

Anche in questo caso, per ricavare l'equazione della parabola bisogna impostare un sistema matematico, che in questo caso conterrà:

  1. L'equazione dell'asse;
  2. Le coordinate dei punti sostituite nell'equazione canonica, esplicitando solo i parametri.

Lavoriamo con un esempio per capire meglio: abbiamo il punto P (-1;1), il punto Q (0;4) e l'asse di simmetria. Il sistema impostato sarà:

    P { a - b + c = 1

    Q { c = 4

    asse { -b/2a

Risolvendo il sistema si troveranno tutti i parametri e l'equazione sarà completa.

3. Noti 3 punti

Conoscendo le coordinate di 3 punti appartenenti alla parabola, si apre un sistema come il precedente, dove le coordinate dei punti sono sostituite nell'equazione generica esplicitando solo i parametri.

Lavoriamo sempre con un esempio: abbiamo il punto A (2;0), B (4;1) e C (12;2). Il sistema impostato sarà:

    A { c = 2

    B { a + b + c = 4

    C { 4a + 2b + c = 12

Risolvendo il sistema si giungerà a conoscere tutti i parametri e avendo dunque l'equazione completa.
