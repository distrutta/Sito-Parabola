Parabola - Definizione

La parabola è il luogo geometrico di tutti i punti del piano equidistanti da un punto detto fuoco e una retta detta direttrice .

La parabola ha equazione del tipo y = ax² + bx + c . Se a < 0 la parabola volge la concavità verso il basso, se a > 0 volge la concavità verso l'alto. Se b = 0, la parabola è simmetrica rispetto all'asse y. Se c = 0 la parabola passa per l'origine.

Le intersezioni della parabola con l'asse x sono le sue soluzioni; esse si ricavano attraverso la formula quadratica: x = [-b ± √ (b² - 4ac)] / (2a) .

Il numero delle soluzioni è dato dal delta Δ (formula: Δ = b² - 4ac). Se Δ < 0 non ci sono soluzioni reali; se Δ = 0 esiste una soluzione; se Δ > 0 esistono 2 soluzioni reali.

Segue qui una tabella che contiene i punti notevoli della parabola e le loro formule:

Punto Descrizione Formula
Vertice Il punto estremo della parabola V = (-b/2a ; -Δ/4a)
Fuoco Fuoco della parabola F = (-b/2a ; (1 - &Delta)/4a)
Asse Asse di simmetria x = -b/2a
Direttrice Retta direttrice y = (-1 + Δ)/4a
